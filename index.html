首先下载安装node.js

官网：https://nodejs.org/en/，下载之后一直下一步无脑安装即可

安装完成可以用打开cmd检验一下是否安装成功，用 node -v 和 npm -v 命令检查版本，如下

安装cnpm 在cmd命令行输入命令

输入 cnpm -v 检查是否安装成功

接下来输入下面的命令，安装hexo

然后输入hexo -v 检查是否安装成功

hexo安装成功后，需要新建一个文件夹，用来放我们的博客，我反在D:\BaiduNetdiskWorkspace\myblog目录下。

新建之后进入到该目录下输入命令 hexo init 初始化博客

之后输入命令 hexo s 启动我们的博客

浏览器输入 http://localhost:4000 就可以打开新部署的网页：

至此博客的搭建已经完成了，但是这只是在本地启动的。还需要部署到远端，我们可以部署到Github上面
